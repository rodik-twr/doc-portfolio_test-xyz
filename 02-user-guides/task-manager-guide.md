 # Task Manager — User Guide

  **Version:** 1.0
  **Last updated:** April 2026
  **Audience:** End users with no prior experience

  ---

  ## Overview

  Task Manager is a web application that helps teams track and prioritize work items.
  This guide covers the core features: creating tasks, assigning them, and tracking progress.

  ---

  ## Getting Started

  ### Prerequisites
  - A valid user account (contact your administrator)
  - A supported browser: Chrome 110+, Firefox 108+, Edge 110+

  ### Logging In

  1. Open your browser and go to `https://taskmanager.example.com`
  2. Enter your **email address** and **password**
  3. Click **Log In**

  > **Note:** If you forget your password, click **Forgot password?** on the login page.

  ---

  ## Creating a Task

  1. Click **+ New Task** in the top-right corner
  2. Fill in the required fields:
     - **Title** — brief description (max 100 characters)
     - **Assignee** — select a team member from the dropdown
     - **Due date** — click the calendar icon to select a date
  3. Optionally add a **Description**
  4. Click **Save**

  The task appears in the **Backlog** column of the board.

  ---

  ## Task Status

  | Status | Meaning |
  |---|---|
  | Backlog | Task created, not started |
  | In Progress | Assignee is actively working |
  | In Review | Work done, waiting for approval |
  | Done | Task completed and accepted |

  ---

  ## Troubleshooting

  **I cannot log in.**
  Verify your email and password. If the issue persists, contact your administrator.

  **A task I created has disappeared.**
  Check the **Archive** section — tasks older than 90 days are archived automatically.
